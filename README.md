# QUIZD4MI2025
QuizMI - Aplikasi Kuis Manajemen Informatika Sebuah aplikasi kuis sederhana yang dibangun menggunakan Flutter, ditujukan untuk mahasiswa baru (maba) program studi D4 Manajemen Informatika. Aplikasi ini dirancang sebagai sarana perkenalan dan pengujian pemahaman dasar seputar dunia IT dan komputer dengan cara yang menyenangkan.
